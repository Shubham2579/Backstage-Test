# Migration Assistant Eclipse Plugin

The Eclipse Plugin provides a developer interface for running modernization assessments directly inside Eclipse IDE.

## Features

- Trigger code assessments
- Display modernization reports
- Integrate with AI analysis
- Export assessment results


## Technology

- Eclipse RCP
- SWT UI
- Maven integration
- Git integration
- HTTP client communication with backend services

## Dependencies

Internal services:

- assessment-service
- genai-service
- template-config

## Usage

1. Install plugin in Eclipse
2. Open a Java project
3. Run **Migration Assessment**
4. View generated modernization report
