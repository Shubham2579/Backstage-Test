# GenAI Service

GenAI Service provides AI-powered capabilities for analyzing codebases and generating modernization recommendations.

It is used by the Assessment Service to enhance analysis results with AI insights.

## Responsibilities

- AI-driven code analysis
- Migration recommendations
- Pattern detection
- Code improvement suggestions

## Technology

- Spring Boot
- REST APIs
- JSON processing (Jackson, Gson)

## Integration

This service is called by: assessment-service
