# Assessment Service

Assessment Service is the core backend component of the Migration Assistant platform.

It analyzes source code repositories and generates modernization assessment reports.

## Responsibilities

- Analyze source code structure
- Generate modernization reports
- Integrate with GenAI Service
- Provide REST APIs for plugins and external clients


## Key Features

- Codebase analysis
- AI-assisted modernization recommendations
- Report generation using templates
- REST API endpoints

## Dependencies

Internal modules:

- genai-service
- template-config

External frameworks:

- Spring Boot
- Spring Security
- Jackson
- FreeMarker
- Playwright automation
